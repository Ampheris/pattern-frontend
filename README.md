## Get started with developing
* Download repo
* Install requirements:
    ```text
    composer install
    npm install
    ```
* Run development server:
    ```text
    php artisan serve
    ```
* To compile css and js:
	```text
    npm run dev
    ```
    or
	```text
    npm run watch
    ```

## Code quality
Make sure you keep a good code standard when contributing
